# Hellobody 💪🔥

## What is Hellobody?? 🤔

헬로바디(Hellobody)는 같은 지역(League)에 거주하는 가입자들 간의 운동 기록을 저장하고, Tier 시스템을 통해 기록을 경쟁할 수 있도록
만든 웹사이트입니다.

회원들은 마치 SNS처럼 본인들의 기록을 저장할 수 있으며, 일정 수의
좋아요(Respect)를 받은 게시물의 경우 자신의 포토카드에 뱃지로 등록할
수 있습니다.
